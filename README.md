# 🏋️ Gym Tracker App

A simple Python application that helps users track their workouts, store progress, and visualize performance over time.

---

## 🚀 Features

- Create a workout plan (reps + sets goals)
- Log completed workouts
- Store workout history using JSON
- Visualize progress using graphs (matplotlib)

---

## 🧠 How it works

1. User creates a workout plan
2. Inputs exercises, reps, and set goals
3. Logs completed workout
4. App compares performance vs goals
5. Data is saved and graph is generated

---

## 🛠️ Technologies Used

- Python
- JSON (data storage)
- Matplotlib (graphs)

---

## ▶️ How to Run

1. Install Python
2. Install matplotlib:

   pip install matplotlib

3. Run the app:

   python gymapp.py
